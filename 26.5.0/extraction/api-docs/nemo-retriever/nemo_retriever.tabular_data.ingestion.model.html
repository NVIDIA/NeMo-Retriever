get_column_ids() → list[str][source]#

Return deduplicated IDs of Column nodes linked by this query’s edges.

In the flat edge structure produced by parse_query_slim, Column nodes are the leaf nodes — they have no further outgoing edges. This mirrors the old get_leafs_from_graph heuristic used for pre-filtering duplicate candidates.

Total number of nodes in the sqlglot AST.

Set by parse_query_slim after extract_tables_and_columns parses the SQL. Acts as a cheap structural fingerprint for pre-filtering duplicate candidates — two queries with different AST sizes cannot be structurally equivalent.
