Run graphic-elements + OCR on chart crops and produce structure-aware text.

For each row (page) in batch_df: 1. Read page_elements_v3 detections and page_image["image_b64"]. 2. Crop all chart detections from the page image. 3. Run graphic-elements model on each crop to get element bboxes. 4. Run OCR on each crop to get text with bboxes. 5. Join the two outputs using join_graphic_elements_and_ocr_output()

to produce semantically structured chart text.

  1. Fall back to OCR-only text if graphic-elements returns no detections.

Returns:

Original columns plus chart and graphic_elements_ocr_v1.

Return type:

pandas.DataFrame
