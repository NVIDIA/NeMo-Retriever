Reads a query CSV, embeds each query, searches LanceDB, prints top-k results, and prints recall@1/@5/@10.

Note: recall@10 requires retrieving at least 10 results per query; we will query LanceDB with search_k = max(top_k, 10) but only print the first –top-k.
